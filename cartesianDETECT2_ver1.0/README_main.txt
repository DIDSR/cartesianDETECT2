This package contains two folders - cartesianDETECT2 and Additional_files_DOIestimation.  

cartesianDETECT2 contains source code, examples and manual for compiling and executing cartesianDETECT2, a Monte Carlo tool for modeling pixelated scintillator detectors.

Additional_files_DOIestimation contains codes and sample data files for estimating the depth of interaction in a scintillator detector using cartesianDETECT2.

Both the folders contain their respective manuals or readme files.
